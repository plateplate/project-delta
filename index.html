ClickLess

Fixing tiny inputs so you click less — demo: month/year entry

Before After About / README

Before — the inefficient UI

This panel mimics a long dropdown/scroll wheel where selecting a month or year requires many clicks/scrolls.

This is slow: many options and dropdowns cause extra clicks.

After — improved prototype

A compact month grid and quick-year control reduce clicks. Buttons are large, keyboard-focusable, and labelled for accessibility.

Quick month and year picker
Click month then year. Works with keyboard. Faster and fewer clicks.

About / Instructions

Open DevTools Console to see metrics. Interact with BEFORE and AFTER panels. Compare click counts and times.

  • Repo: Put source files in the repo root (index.html, styles.css, app.js).
  • Deploy: GitHub Pages from `main` branch or `gh-pages` branch.
  • Accessibility: Run WAVE and axe on your deployed URL.

View source code in this repo once uploaded to GitHub.
